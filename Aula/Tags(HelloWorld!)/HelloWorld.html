  • <section> - Define uma seção em um documento.Exemplo

Novas Tags:

  • <base> - Especifica a URL base/target para todas as URLs relativas em uma página.Exemplo
  • <link> - Define a relação entre o documento atual e um recurso externo (como uma folha de estilo).Exemplo
  • <meta> - Fornece metadados sobre o documento HTML, como descrições, palavras-chave e informações de codificação.Exemplo
  • <address> - Define informações de contato para o autor/proprietário de um documento ou artigo.Exemplo
  • <article> - Define um conteúdo independente que faz sentido por si só, como um post de blog ou uma notícia.Exemplo
  • <asidegt; - Define um conteúdo relacionado ao conteúdo principal, como uma barra lateral em um layout.Exemplo
  • <nav> - Define uma seção de navegação em um documento.Exemplo
  • <dd> - Define uma descrição em uma lista de definição dentro de um </li>Exemplo
  • <dl> - Define uma lista de definição, que consiste em pares de termos e suas definições </dt>Exemplo

Oque é HTML

*HTML significa linguagem de marcação de hipertexto.

*HTML é a linguagem de marcação padrão para criação de página Web.

*HTML descreve a estrutura de um apágina Web.

*HTML consiste em uma série de elementos.

*Os elementos de HTML informam ao navegador como exibir o conteúdo.

*Os elementos HTML rotulam partes do conteúdo como "este é um título", "este é um parágrafo", este é um link", etc.
